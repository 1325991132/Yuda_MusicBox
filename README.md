# YD Music Box (vue3)

### Demo演示网站，使用网易云音乐账号登录，开启属于你自己的音乐盒
```
http://www.qyd123.cn
```

### 后台api来自网易云音乐，node后台工程文件中的wyapi文件夹中,进入该目录下，可运行后台接口
```
yarn install
yarn start
```
### 前端使用vue3
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
